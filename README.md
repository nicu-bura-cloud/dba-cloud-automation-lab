DBA Cloud Automation Lab

Laboratorio pratico per imparare PostgreSQL, Terraform e Ansible.

Obiettivi

* Creare infrastruttura AWS con Terraform
* Installare PostgreSQL con Ansible
* Gestire database e tabelle PostgreSQL
* Eseguire backup con pg_dump
* Eseguire restore da backup
* Comprendere Foreign Key e JOIN
* Comprendere transazioni BEGIN, COMMIT e ROLLBACK

Tecnologie

* AWS EC2
* Terraform
* Ansible
* PostgreSQL
* Linux Ubuntu

Attività svolte

* Deploy di una EC2 con Terraform
* Installazione automatica di PostgreSQL
* Creazione database labdb
* Test backup e restore
* Test Foreign Key
* Test LEFT JOIN
* Test transazioni e isolamento
